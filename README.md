# MinuteGram

MinuteGram is a streamlined social platform that organizes content into topics and posts. Users can register and authenticate, then browse topics and create posts with text, audio or video. Every post and comment can be liked, and users may reply to posts or to existing comments to foster conversation.

## Tech Stack

The backend is implemented in Java with Spring Boot. The frontend is built using React framework. Data is persisted in PostgreSQL.
